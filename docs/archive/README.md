# Архив документов

В этой папке лежат документы, которые описывают историю внедрения, планы этапов или разовые аудиты.
Они **не являются источником текущих runtime-правил**.

## Что здесь хранится

- phase/roadmap документы (`PHASE_*`, `M*`, `MIGRATION_*`, `MAP`, `ARCH_GAP_ANALYSIS`);
- исторические сценарии и заметки внедрения.
- архивные планы/аудиты:
  - `AGENT_MINIPROMPTS.md`
  - `MEMORY_COMPANION_MINIPROMPTS.md`
  - `PHASE_0_FINDINGS.md`
  - `API_CONTRACT.md` (старый контракт OpenWebUI)

## Как использовать

- Для актуальных правил и фактического поведения смотри:
  - `../DevRules.md`
  - `../Architecture.md`
  - `../PROJECT_OVERVIEW.md`
  - `../COMMANDS.md`
  - `../CONTRIBUTING.md`
  - `../MWV_CANONICAL_FLOW.md`
  - `../RUNTIME_MWV_FLOW.md`
- Документы из `docs/archive/` читать только как исторический контекст.
